# rust_study
for study

## how to read
- vscode 기준 rust-analyzer 설치 필수
- 폴더 추가시 루트 Cargo.toml에 member 추가.
- cargo 미사용 프로젝트는 exclude 추가. 
- 각 소스에서 run & debug 가능.
